# 企画書 — DaD（仮称：Dungeon and Darker）

> 本書はユーザーからの最初の指示を企画書として保管したものです。以降の開発はこの仕様を基準とします。
> 作成日: 2026-05-30

---

## 1. コンセプト

2D俯瞰視点（ただし壁の高さ・影で立体感を出す擬似3D表現）の、**Dark and Darker** に着想を得たエクストラクション型ダンジョンRPG。

- 収集・装備したアイテムを持ってダンジョンへ潜入
- 敵と戦いながら戦利品（ルート）を獲得
- **生きて脱出口から帰還すれば持ち帰り成功**
- **死亡したら持ち込み・取得したアイテム/装備はすべてロスト**
- 職業制（クレリック・メイジ等の王道ファンタジー職）
- 持ち帰った戦利品は**売却**してゴールド化できる
- キャラクターを**育成**できる（レベル・ステータス成長）

## 2. プラットフォーム / 操作

- **スマホ横持ち（ランドスケープ）** 前提。PCブラウザ（キーボード）でも操作可。
- **左ジョイスティック**：移動（360°）
- **右側スキルボタン**：スキルを選択
- **右ジョイスティック**：攻撃方向を決めて放つ（離した瞬間 or ドラッグ方向に発射）
- **スキル未選択で右スティックを使った場合 → ノーマル武器攻撃**
  （剣／弓／杖（魔法弾）／ハンマー 等、職業の標準武器による攻撃）
- HP / MP 回復アイテムをスロットから使用

## 3. 視界・ライティング（重要システム）

1. **方向視界**：キャラの向いている方向は明るく見え、**真反対（背後）は暗くて見えない**（視界コーン）。
2. **暗いマップ**：松明（トーチ）が無いと視界半径が狭い。トーチ装備で視界が広がる。
3. **部屋単位のフォグ**：プレイヤーがいる部屋以外は暗くて隠れる。
   - **扉を開ける／その部屋に入ると明るくなって見える**（=その部屋がリビール）。
4. マップ上の固定光源（壁トーチ・焚き火等）はその周囲を照らす。

## 4. マップ

- 広めのダンジョン。**複数の部屋**＋それを繋ぐ通路・扉。
- 部屋ごとに明るさ（ambient）を持つ。脱出ポータルを配置。
- ランダム生成（プレイのたびに変化）。

## 5. 職業（クラス）

王道の構成。各職に**固有の標準武器**と**専用スキル**、**ステータス傾向**を持たせる。

| 職業 | 標準武器 | 特徴 |
|---|---|---|
| ファイター | 剣 | バランス型・近接万能 |
| バーバリアン | ハンマー/斧 | 高HP高火力・低速の脳筋近接 |
| レンジャー | 弓 | 遠距離物理・カイト戦法 |
| ローグ | 短剣 | 高速・隠密・会心特化 |
| クレリック | メイス＋聖法 | 回復・支援・対アンデッド |
| メイジ（ウィザード） | 杖（魔法弾） | 高火力魔法・低耐久・MP管理 |

各職共通：**スキル未選択時の右スティック＝標準武器でのノーマル攻撃**。

## 6. ステータス制度（細かく）

基礎ステータス（一次パラメータ）:

- **STR（筋力）**：近接物理ダメージ、所持重量
- **VIG/CON（体力）**：最大HP
- **AGI（敏捷）**：移動速度、攻撃速度、回避
- **DEX（器用）**：遠距離物理ダメージ、命中、会心率
- **WILL（知力/意志）**：魔法ダメージ、最大MP
- **WIS（精神）**：回復力、MP回復、魔法耐性
- **LUCK（幸運）**：レアドロップ率、会心ダメージ

派生（二次）ステータス:
- 最大HP / 最大MP / 物理攻撃 / 魔法攻撃 / 防御 / 移動速度 / 攻撃速度 / 会心率 / 会心ダメージ / 回避 / 視界半径 / 所持重量

**育成**：ダンジョン帰還・敵撃破でEXP獲得 → レベルアップで**ステータスポイント**を割り振り。装備でも各ステータスが増減。

## 7. アイテム / 装備

- レアリティ：Poor / Common / Uncommon / Rare / Epic / Legendary（色分け）。
- 種別：武器・防具（頭/胴/手/脚/装飾）・トーチ・消費（HP/MPポーション）・財宝（売却用）・素材。
- 装備でステータス補正。ランダム生成（接頭/接尾アフィックス）。
- **インベントリ／装備画面**、**ストレージ（金庫=ロストしない保管）**、**ショップ（売買）**。
- 持ち込み品はダンジョンに持って入る＝死ねばロスト。金庫の品は安全。

## 8. 敵（さまざまに）

- スケルトン（近接）、スケルトンアーチャー（遠隔）、ゾンビ（低速高HP）、スライム（分裂）、
  コウモリ（高速）、ゴブリン、魔導士（遠隔魔法）、ボス（各階層の主）等。
- AI：徘徊→索敵→追跡→攻撃。視界・音で反応。ドロップテーブルを持つ。

## 9. ゲームループ

```
タウン（拠点）
  ├ キャラ作成/選択（職業選択）
  ├ ステータス育成（ポイント割り振り）
  ├ インベントリ/装備
  ├ ショップ（売買）
  ├ 金庫（保管）
  └ ダンジョンへ出撃（持ち込み品を選択）
        │
        ▼
   ダンジョン探索（戦闘・ルート・視界システム）
        │
   ┌────┴────┐
 脱出成功      死亡
 （持ち帰り）  （ロスト）
        │
        ▼
   リザルト → タウンへ
```

## 10. Dark and Darker 参考の要素（調査反映）

- **エクストラクション**：脱出ポータルから抜けて初めて戦利品が確定。
- **ハイリスク**：死亡で全ロスト（持ち込み＋取得分）。
- **インベントリのグリッド管理**と重量。
- **ライト/視界の戦略性**（トーチ＝光源だが自分の位置を晒す）。
- **クラス＋パーク/スキル構成**。
- **金庫（ストレージ）と市場（売却）経済**。
- **モンスター＋宝箱＋トラップ**のダンジョン構造。
- （将来）ゾーン収縮（毒/闇の侵食）で長居を許さない圧。

## 11. 技術方針

- **HTML5 + Canvas2D + Vanilla JS**（ビルド不要、GitHub Pages にそのまま配置可能）。
- セーブは `localStorage`（キャラ・金庫・ゴールド・育成）。
- タッチ（仮想ジョイスティック2本＋ボタン）＋キーボード両対応。
- 横向き固定、レスポンシブにスケーリング。

## 12. 開発フェーズ

- **Phase 1（今回）**：GitHub で遊べる動くプロトタイプ。コア（移動・視界・部屋フォグ・戦闘・スキル/ノーマル攻撃・敵・ルート・脱出/ロスト・ステータス・育成・ショップ/金庫・UI）。
- Phase 2：アフィックス強化、ボス、トラップ、サウンド、バランス調整、職業追加。
