# 輪廻 -RINNE-

> 旧称 Dungeon & Darker。仏教の六道・十界をモチーフに全面リニューアル。

六道・十界をさまよう、2D俯瞰（ドット絵）のエクストラクション型ダンジョンRPG。
深く潜るほど人間界 → 修羅界 → … → 無間地獄へと堕ち、難易度と戦利品のレア度が上がる。
死ねば全てを失い、また人間界へ還る——輪廻。

## （旧称ドキュメント）⚔️ Dungeon & Darker

**Dark and Darker** に着想を得た、2D俯瞰（擬似3D）のエクストラクション型ダンジョンRPG。
スマホ横持ち向けのブラウザゲームで、**GitHub Pages で URL を開くだけ**で遊べます。

> 装備を持ってダンジョンへ潜り、敵を倒して戦利品を集め、**生きて脱出**せよ。
> **死ねば持ち込んだ全てを失う。**

## ▶ 遊ぶ

GitHub Pages を有効にすると下記で公開されます：

**https://tmk4men.github.io/dungeon-and-darker/**

（PCブラウザでもプレイ可。スマホは横向き推奨。）

## 🎮 操作

| | スマホ（横持ち） | PC |
|---|---|---|
| 移動 | 画面左半分をドラッグ（仮想スティック） | WASD / 矢印キー |
| 攻撃方向＆発射 | 画面右半分をドラッグ→離して発射 | マウス照準＋スペース／クリック |
| スキル選択 | 右下のスキルボタン | 1 / 2（0で通常に戻す） |
| 回避ローリング | 中央下の「回避」ボタン | Shift |
| ポーション | 右下のポーションボタン | Q / E |
| 祭壇/聖域 | 近づくと出る「祈る/捧げる」ボタン | F |

**スキルを選ばずに右スティックを使うと、職業の標準武器による通常攻撃**になります。

## ✨ 実装済みの要素

- **職業9種**：ファイター / バーバリアン / レンジャー / ローグ / クレリック / メイジ / **パラディン / ウォーロック / ランサー**（固有の標準武器＋専用スキル）
- **細かいステータス制度**（STR/VIG/AGI/DEX/WILL/WIS/LUCK）＋**育成**（EXP・レベル・ポイント割り振り）
- **視界システム**：方向視界コーン（背後は暗い）／トーチで視界拡大／**部屋単位のフォグ**（扉を開ける・入室で明るくなる）
- **擬似3D**の壁・影表現
- **広いランダムダンジョン**（複数の部屋＋通路＋扉＋光源＋脱出ポータル）
- **トラップ**：床スパイク（予兆付き周期発動）／ダート発射トラップ
- **ゾーン収縮（闇の侵食）**：時間経過で安全圏が縮小、圏外はダメージ → 長居は禁物
- **多様な敵（13種＋ボス3種）**：スケルトン、弓兵、ゾンビ、スライム（分裂）、コウモリ、ゴブリン、ダークメイジ、大蜘蛛（鈍足付与）、スケルトン騎士、インプ、幽鬼／ボス（リッチ王・オーガ・**ネクロマンサー＝死霊召喚**）
- **状態異常**：スタン／ライフスティール／毒(DoT)／鈍足
- **サウンド**：WebAudio によるSFX合成（攻撃・被弾・スキル・トラップ・脱出など。🔊ミュート可）
- **エクストラクション**：脱出で戦利品確定／**死亡で全ロスト**
- **アイテム/レアリティ**（6段階）・アフィックス・装備によるステータス補正
- **HP/MPポーション**
- **経済**：ショップ（売買）／倉庫（安全な保管）／ゴールド
- **こだわりUI**（タウン・育成・装備・倉庫・ショップ・HUD・リザルト）

### さらに（Phase 3）
- **戦闘の手触り**：回避ローリング（無敵フレーム）／敵の予備動作（テレゴラフ＝見て避ける）／ヒットストップ＋画面シェイク／**エリート敵**（迅速・巨躯・灼熱・猛毒・再生・魔導）
- **ビルド深化**：**スキル選択制**（職業ごと4種から2つ装備）／装備の**強化＋エンチャント**（鍛冶）／**吸血**ステータス／**レジェンダリ固有効果**／装備比較UI
- **装備重量 → 移動速度**（STRで許容重量が増え、超過で減速）
- **リスク/報酬**：**複数の脱出口**（時間で開く★報酬ポータル）／**特殊部屋**（生贄の祭壇＝HPと引き換えに戦利品／聖域＝恩恵ガチャ）／**バイオーム**（古城・洞窟・氷窟・溶岩窟）
- **メタ進行**：**ミニマップ**／**依頼（バウンティ）**／**実績**／**AIライバル冒険者**（敵と戦い戦利品を奪い合うPvPvE風味）

詳しい仕様は [`企画書.md`](企画書.md) を参照。
